- `identified`: desire identified
- `samskara`: has samskara chain complete

- `solo`: Not in any thread
- `threaded()`: Part of a thread
- threadId?: string
- `klesha`: Marked as problematic
- `tainted`: Downstream from klesha
- `completed`: Task finished
